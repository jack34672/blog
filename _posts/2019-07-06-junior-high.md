畢業五年了，被國中同學揪唱歌。<br>
原本沒有很想去，因為主揪的是我超爆久沒講話，但國中很好的女森。<br>
而且重點是她學期初突然密我好久不見，然後幹一幹(話)我就先已讀她一次，被她隔一個月說「欸我發現你已讀我欸(哭笑臉*3)」又不讀她兩個月，超扯幹我真的超壞。<br>
但我跟她國中真的還不錯，因為我好像滿喜歡她的。<br>
反正就被另一個一直都有聯絡的國中同學約就只能答應了。<br><br>

但好懷念哦，國中的感覺，年輕的感覺。<br><br>

講起國中的事，雖然我大概忘了90%，但會有些印象，童年的印象，發現我國中其實過得很開心。雖然我不是班核，也不是男生的風雲人物，但我跟大家都還不錯。<br>
國中男生有兩個89，跟女生都混很好，然後各跟一兩個女生交往，但我就超討厭他們，因為他們對人就是有種高人一等的感覺。<br>
我就很不解為啥女生會喜歡那種，根本低能。<br>
班上又有幾個女生很愛搞事，誰喜歡誰，誰說誰壞話，問說你是哪一派的，不小心就哭了之類的。然後我就會嘴砲這些低能兒，所以我也跟她們不好。<br>
但今天去的就都是正常女生，反正我也很久沒唱歌了，就唱一下。<br><br>

整個的感覺就很舒服，不用因為是現在朋友就有些矜持，反正一年就見這幾次，又因為都認識那麼久，也不需要有些收斂。<br>
聊起國中的事就像很多trigger一樣，勾起年輕的回憶。我現在真的心態超老。<br>
然後發現我唱KTV好像還蠻屌，美秀屌唱一波。<br>
唱完還去圓山舊兒童樂園探險。好玩。又約了一波酒吧。<br><br>

我的國中就是個讓我學習一下男女相處跟朋友打幹的階段。<br>
我還記得那時候會跟女生勾肩搭背，分食東西，然後有時候抱一下之類的。<br>
然後就會微爽一下，就算不是我喜歡的那個女生。<br>
我還記得我國中畢旅時屌虐，有個女生一直叫我去搭馬雅探險，我就不敢，我就說不然你給我牽我就搭，然後我就搭了，我整場都在覺得她手好好摸，有點變態但真的還蠻好摸阿幹。<br>
總而言之就是個友善的交流環境，所以我還蠻快樂的。<br>
然後這次發現就是有個同學她味道居然一樣，就每個人都會有個味道，我鼻子又很猛都聞得到，我超爆懷念的。因為我好像國中很喜歡她。<br>
還記得國中時會生日跟她交換禮物之類的，然後收到卡片就會暗爽，然後傳簡訊(so fking old)就很開心，但就不知道為甚麼從來不會表態一下。<br>
但現在看一看發現我之後交的都比較正啦ㄏㄏ。<br>
我可以沒辦法跟比較平庸的人相處了，除了很熟的國中同學以外。<br><br>

這次微微遺憾就是，沒有問一些國中八卦，當時在我心中疑惑很久，她到底喜歡誰，有沒有喜歡臭89，就可以知道她到底膚不膚淺，但又為甚麼跟我真的還不錯，是因為我功課屌虐我工具嗎？<br>
但最後就是沒問，可能是怕問了太怪，但最可能是因為我現在根本不在乎了。<br><br>

只要曾經享受過彼此的陪伴就好了吧，彼此 means 同學們，過程或細節在幾年後根本不重要了，重要的只是這份熟悉的感覺。<br><br>

知道每個長假都要約一下，講講這半年來的近況，抱怨或分享事情，不需要面具，不需要隱藏，因為彼此沒有目的，沒有需求，就是純粹的友誼。<br><br>

雖然我國中的事都忘記了，但記得這些朋友，在我平常不爽或不知道跟誰說話時可以有個出口。真的滿珍惜這種純粹的友誼的。<br><br>

function slugify(text) {
  return text.toString().toLowerCase().trim()
    .replace(/&/g, '-and-')
    .replace(/[\s\W-]+/g, '-')
    .replace(/[^a-zA-Z0-9-_]+/g,'');
}


function escapeHtml(str) {
    var div = document.createElement('div');
    div.appendChild(document.createTextNode(str));
    return div.innerHTML;
}

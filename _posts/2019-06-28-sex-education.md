<div>
花了三天把性愛自修室看完。來寫個心得。<br><br>

看完結局後心情是好的，但也沒有到非常好，畢竟不是理想中Otis跟Maeve在一起。<br>
整體劇情滿完整且有連貫性，用許多小配角的插曲來描述主角的人格。<br>
笑點真的幽默，有點情色又不失水準的橋段頗多，讚。<br><br>
雖然知道這應該是校園喜劇的誇張版，很多歧視/性愛成分都有點過度誇張，(我就不相信有人像Aimee一直被利用還願意待在那個團體==)<br>
但就，還蠻好笑的，又有講出一些道理。<br>
有些青少年的疑惑/迷茫/困擾，都在劇裡有些「乾我好像有這樣」的感覺，雖然我已經20歲了。<br><br>
突然發現自己20歲又覺得好老，已經不是teenager了，怕爆。<br><br>
總而言之，如果Game of Thrones的前幾季有5/5的話，Sex Education應該有4.2/5吧。<br><br>
</div>

function slugify(text) {
  return text.toString().toLowerCase().trim()
    .replace(/&/g, '-and-')
    .replace(/[\s\W-]+/g, '-')
    .replace(/[^a-zA-Z0-9-_]+/g,'');
}


function escapeHtml(str) {
    var div = document.createElement('div');
    div.appendChild(document.createTextNode(str));
    return div.innerHTML;
